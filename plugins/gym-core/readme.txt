Core functionality for Haanpaa Martial Arts — a WooCommerce-powered martial arts gym management system.

== Description ==

Gym Core is the foundation plugin for the Haanpaa Martial Arts WooCommerce site. It powers:

* **Multi-location support** — taxonomy and data structures for the Rockford and Beloit locations
* **Membership integration** — WooCommerce-based membership tiers and access control
* **Class schedule system** — schedule management for instructors and students
* **Belt rank tracking** — promotion records, rank history, and requirements
* **Attendance & check-in** — per-class attendance logging for members
* **Gamification** — badges, streaks, and milestone rewards
* **Twilio SMS** — opt-in notifications for class reminders and promotions
* **REST API** — authenticated endpoints for AI agent integrations

== Installation ==

1. Ensure WooCommerce 10.3+ is active.
2. Upload the `gym-core` folder to `/wp-content/plugins/`.
3. Activate the plugin through **Plugins > Installed Plugins**.
4. Configure settings under **WooCommerce > Gym Core Settings**.
