# Tool Mapping -- Where Things Live Now

A quick-reference for staff during the transition. Bookmark this page. When you
catch yourself reaching for an old tool, check the table below to find where
that workflow lives in the new system.

---

## Spark Membership --> New System

| What You Did | Where in Spark | Where Now | How to Get There |
|---|---|---|---|
| View all members | Members list | Jetpack CRM contacts | Gym CRM in wp-admin sidebar |
| Search for a member | Members > Search | CRM search bar | Type name, email, or phone |
| View member details | Click member name | CRM contact record | Belt rank, classes, membership auto-synced |
| Check in a member | Check-in tab / app | Gym > Attendance > Today | Quick Check-In form or kiosk |
| View attendance history | Reports > Attendance | Gym > Attendance > History | Filter by date, class, student, location |
| Track belt ranks | Ranks section | Gym > Promotions | Eligibility auto-calculated |
| Promote a student | Edit member rank | Gym > Promotions > Promote | Click Promote, select new rank |
| Manage memberships | Billing > Plans | WooCommerce > Subscriptions | View, pause, cancel, retry payments |
| Create a membership | Add member + plan | WooCommerce > Orders > Add New | Or member self-serves on website |
| Process a payment | POS or billing | WooCommerce > Orders | Automatic for subscriptions |
| Send SMS to member | Spark SMS | Gandalf AI or direct Twilio | Ask Sales/Admin agent to draft |
| View revenue reports | Spark Reports | WooCommerce > Analytics > Revenue | Or ask the Finance agent |
| Manage class schedule | Schedule section | Gym > Classes | Add, edit, or cancel classes |
