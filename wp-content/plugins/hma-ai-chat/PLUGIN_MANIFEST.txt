CORE FILES
==========
hma-ai-chat.php
  - Main plugin bootstrap
  - Defines constants: HMA_AI_CHAT_VERSION, HMA_AI_CHAT_FILE, HMA_AI_CHAT_PATH, HMA_AI_CHAT_URL
  - Checks WP 7.0 and WP AI Client availability
  - Registers activation/deactivation hooks
  - Initializes plugin on plugins_loaded

uninstall.php
  - Drops custom tables
  - Deletes plugin options
  - Fires cleanup action

composer.json
  - PSR-4 autoloading for HMA_AI_Chat\ namespace
  - PHP 8.0+ requirement

SRC FILES - PLUGIN CLASS
=======================
src/Plugin.php
  - Main orchestrator class (singleton pattern)
  - Registers all hooks on admin_init
  - Initializes Chat Page, REST routes, Agent Registry
  - Enqueues admin assets

src/Activator.php
  - Creates three custom database tables
  - Initializes webhook shared secret
  - Updates schema version option

src/Deactivator.php
  - Clears scheduled events on deactivation
  - Placeholder for future cleanup tasks

SRC FILES - ADMIN INTERFACE
===========================
src/Admin/ChatPage.php
  - Adds submenu under Tools > AI Chat
  - Renders chat container div
  - Enqueues chat-app.js and chat-app.css
  - Localizes script with:
    * REST API URL
    * Nonce
    * Available agents
    * Current user role
    * UI strings

SRC FILES - REST API
====================
src/API/MessageEndpoint.php
  Route: POST /hma-ai-chat/v1/message
  - Validates agent and message input
  - Checks user capability
  - Gets/creates conversation
  - Stores user message
  - Calls wp_ai_client_prompt() with Claude Sonnet 4.6
  - Stores assistant response
  - Returns: response, conversation_id, tokens_used

src/API/HeartbeatEndpoint.php
  Route: POST /hma-ai-chat/v1/heartbeat
  - Validates webhook signature and IP
  - Accepts: runId, agentId, taskId, wakeReason
  - Handles: approval_needed, execution_complete
  - Creates pending action records
  - Fires hma_ai_chat_execution_complete action

SRC FILES - AGENTS
==================
src/Agents/AgentPersona.php
  - Represents individual agent
  - Properties: slug, name, description, system_prompt, required_capability, icon
  - get_system_prompt() injects dynamic context (date, site name, URL)

src/Agents/AgentRegistry.php
  - Singleton registry for all agents
  - Registers four default agents:
    * sales (💼, edit_posts)
    * coaching (🥋, edit_posts)
    * finance (💰, manage_options)
    * admin (⚙️, manage_options)
  - get_available_agents($user_id) - filters by capability
  - Fires hma_ai_chat_agents_registered action

SRC FILES - DATA PERSISTENCE
============================
src/Data/ConversationStore.php
  Tables: wp_hma_ai_conversations, wp_hma_ai_messages
  Operations:
    - create_conversation()
    - save_message()
    - get_conversation() - retrieves history
    - get_user_conversations() - lists conversations
    - delete_conversation() - with ownership check
    - update_conversation_title()

src/Data/PendingActionStore.php
  Table: wp_hma_ai_pending_actions
  Operations:
    - store_pending_action()
    - get_pending_actions()
    - approve_action() - fires hma_ai_chat_action_approved
    - reject_action() - fires hma_ai_chat_action_rejected
    - get_action()

SRC FILES - SECURITY
====================
src/Security/WebhookValidator.php
  - validate_request() - verifies Bearer token with constant-time comparison
  - validate_ip() - checks against allowlist
  - get_secret() / generate_secret() / rotate_secret()
  - get_ip_allowlist() / update_ip_allowlist()
  - get_client_ip() - supports proxies

DATABASE SCHEMA
===============

wp_hma_ai_conversations
  - Indexed: user_id, agent, created_at
  - ON DELETE CASCADE for messages

wp_hma_ai_messages
  - Indexed: conversation_id
  - Foreign key to conversations table
  - Stores role (user/assistant) and content

wp_hma_ai_pending_actions
  - Indexed: agent, status, run_id
  - JSON-encoded action_data
  - Tracks approval metadata (approved_at, approved_by)

PLUGIN OPTIONS
==============
hma_ai_chat_webhook_secret
  - SHA-256 hash used for Paperclip webhook auth
  - Generated on activation if missing

hma_ai_chat_ip_allowlist
  - Array of IP addresses allowed for webhooks
  - Validated with filter_var(FILTER_VALIDATE_IP)

hma_ai_chat_db_version
  - Tracks schema version for migrations
